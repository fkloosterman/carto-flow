# Tutorials

Step-by-step guides to help you learn carto-flow through hands-on examples and real-world scenarios.

## Getting Started

### [Flow-Based Cartograms](basic-flow-cartogram.ipynb)
Master morphing algorithms, data preparation, and parameter tuning for production-quality cartograms.

### [Proportional Cartograms](basic-proportional-cartogram.ipynb)
Split and shrink shapes to represent proportions and multiple data dimensions within regions.

### [Symbol Cartograms](basic-symbol-cartogram.ipynb)
Create Dorling-style symbol cartograms with physics-based circle placement and tiling layouts.
